# NEET Question Generator

A fine-tuned SLM that generates NEET-style multiple choice questions across Physics, Chemistry, Botany, and Zoology based on previous year question patterns.

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/#create=true&url=https%3A%2F%2Fgithub.com%2Fvinothvikas1987%2FUG_NEET_MOCK_QUESTIONS_AI%2Fblob%2Fmaster%2Fneet_app.py)

## Features

- Generate NEET MCQs for any subject/topic in the official NEET syllabus
- Covers 20 Physics topics, 20 Chemistry topics, 10 Botany topics, 9 Zoology topics
- Trained on real NEET question papers (2019-2025)
- Outputs structured questions with 4 options and correct answer
- Access key protected (share key with users)
- 5 free questions per email, then contact for access

## Limitations

- **Text-only**: Cannot generate or render diagrams/images
- Generates questions based on pattern recognition from past papers
